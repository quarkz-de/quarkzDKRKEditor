# quarkzDKRKEditor
Der Editor für "Das Kleine Rote Kochbuch".

Das Projekt entstand als Übungsprojekt zur Verwendung von Spring4D und einem ORM-Framework.
Im ersten Schritt wurde DORM verwendet und dann auf Marshmallow (in Spring4D enthalten)
umgestellt.
